01. API de IA Generativa

Integração Serverless Híbrida (AWS & Azure)

Este projeto demonstra a comunicação segura e eficiente entre provedores de nuvem distintos para entregar inteligência artificial como serviço.

  • Gateway de API: Exposição de endpoint seguro via AWS API Gateway.
  • Processamento: AWS Lambda atuando como middleware para orquestração de chamadas.
  • Inteligência: Consumo de modelos LLM via Azure OpenAI Service (Foundry).
  • Arquitetura: Design desacoplado focado em baixa latência e escalabilidade.
