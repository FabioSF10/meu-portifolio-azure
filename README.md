## 🚀 O que já foi implementado

### 🏗️ Hospedagem Estática (Blob Storage)
* Configuração de **Storage Account** otimizada para site estático.
* Utilização do contêiner especial `$web` para entrega de arquivos de alta performance.
* Ponto de extremidade primário configurado para acesso público global.

### ⚡ Automação de Deploy (GitHub Actions)
* Pipeline de **CI/CD** configurado no arquivo `main.yml`.
* Sincronização automática via **Azure CLI** sempre que um novo código é enviado para a branch principal.
* Uso de segredos de repositório (`Secrets`) para garantir a segurança da Connection String.

## 🛠️ Stack Tecnológica

* **Cloud:** Microsoft Azure.
* **Storage:** Azure Blob Storage (Static Hosting).
* **DevOps:** GitHub Actions & Azure CLI.
* **Linguagens:** HTML5, CSS3 Moderno, JavaScript.
